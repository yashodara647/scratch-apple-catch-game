# Apple Catch Game 🍎

This is a simple game built using Scratch.

## Game Description
The player moves left and right to catch falling apples.
Each apple increases the score.

## Features
- Arrow key movement
- Score system
- Increasing apple speed
- Simple gameplay

## How to Play
1. Click the green flag
2. Move using arrow keys
3. Catch apples to earn points

## Project File
Download the .sb3 file and open it in Scratch.
